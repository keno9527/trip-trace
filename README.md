# Trip Trace

Trip Trace 是一个中文优先、PC 优先、本地优先的家庭旅行轨迹地图 Web 应用。它用于导入旅行照片，从照片 EXIF 中读取拍摄时间和 GPS 坐标，并通过地图、时间轴和家庭成员标签整理旅行回忆。

当前版本是纯前端 MVP：不需要账号，不依赖后端，不做云同步，也不提供公开分享页。

## 当前功能

- 创建和切换旅行。
- 批量导入本地图片文件。
- 解析照片 EXIF 中的拍摄时间和 GPS 坐标。
- 为导入照片生成缩略图和展示尺寸副本。
- 在地图上展示带坐标的照片点位。
- 按拍摄时间连接带坐标的照片，形成旅行路线。
- 在时间轴中按日期分组展示已导入照片。
- 创建家庭成员标签，并把标签应用到选中的照片。
- 按家庭成员标签筛选地图和时间轴中的照片。
- 使用 IndexedDB 在浏览器本地保存旅行、照片元数据、图片副本和成员标签。

## 不包含的能力

- 用户账号和云同步。
- 多人实时编辑。
- 公开分享页面。
- 原生 iOS 相册访问。
- 实时 GPS 记录。
- GPX、KML 或 GeoJSON 轨迹文件导入。
- 浏览器内全尺寸原图长期归档。
- 离线地图瓦片或自托管地图瓦片。

## 数据与隐私说明

Trip Trace 当前只在浏览器本地保存数据。应用会保存缩略图和展示尺寸图片副本，但不承诺长期保存全尺寸原图；浏览器本地存储可能受到容量限制，也可能被浏览器清理。请在应用外继续保留原始照片备份。

地图展示使用 OpenStreetMap 远程瓦片，浏览地图时浏览器可能请求地图瓦片数据。应用不会上传照片或旅行记录。

- 批量照片导入摘要。

- 地图、时间轴、侧栏和成员标签组件行为。
